## Workflow

1. **Explorer** — Map high-level structure: directories, entry points, key modules, config
2. **Analyst** — Quantify codebase: LOC by language, dependency count, test coverage, complexity hotspots
3. **Architect** — Synthesize findings into an architecture overview with key patterns and conventions

## Coordination
- Explorer and analyst work in parallel (independent concerns)
- Architect synthesizes after both complete
- Output is a concise briefing document, not an exhaustive report
